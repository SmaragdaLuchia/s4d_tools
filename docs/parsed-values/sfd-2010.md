# SFD 2010 - parsed outputs

Parser return values are `pandas.DataFrame` objects unless noted. Column dtypes follow what the parsers assign (mostly `object` / string for XML text; some numeric fields remain strings if the source is free text).

`PINParser.parse_all()` returns two tables: `products`, `price_matrices`.

| `species_group_user_id_agency` | str  | products       |
